What's New In KLatexFormula 4.1

This screen can be opened at all times from the "About" window accessible with the button in expanded mode.

KLatexFormula 4.1 presents a few improvements and minor changes. It now supports the PDF-based and unicode-based LaTeX engines pdflatex, xelatex, and lualatex.

PDF/unicode-based LaTeX equation generation is done via a “user script”: In expanded mode, select Script » select user script "run pdflatex engine".

See the release notes for more information, or check out the release notes for KLatexFormula 4.0 for features introduced in KLatexFormula 4.
