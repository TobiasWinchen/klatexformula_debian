About KLatexFormula

Version:

KLatexFormula was written by Philippe Faist, currently (2014) Ph.D. student at ETH Zürich (Switzerland). First released in 2006 written in KDE3 environment, the program has seen many improvements and restructuring since then, leading to the current version, written in Qt 4.
