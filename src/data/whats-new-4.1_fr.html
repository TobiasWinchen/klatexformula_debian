Nouveautés de KLatexFormula 4.1

Cette fenêtre peut être ouverte en tous temps depuis l'affichage « A propos de KLatexFormula » accessible avec le bouton en mode étendu.

KLatexFormula 4.1 présente diverses améliorations, entre autres la génération de PDF directe via les moteurs LaTeX pdflatex, xelatex, et lualatex.

La génération directe de PDF se fait via un « script utilisateur »: En mode étendu, selectionner script ▸ "run pdflatex engine".

Pour plus d'informations, consultez la page d'informations sur les nouveautés de la version 4.1 et les nouveautés de la version 4.
