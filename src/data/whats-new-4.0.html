What's New In KLatexFormula 4.0

This screen can be opened at all times from the "About" window accessible with the button in expanded mode.

KLatexFormula was ported from Qt4 to Qt5, with support for high-resolution screens such as Retina displays.

Some new formats are available, as part of the redesigned data export infrastructure: SVG via dvisvgm or Inkscape, EMF/WMF Windows Metafile via Inkscape, HTML with inline image data, and AICB for Adobe InDesign.

Open the original example library under the library browser › Resources › Open Example Library. Includes equation examples as well as a Feynman diagram.

Several infrastructure changes shouldn't have any greatly noticable effect upon simple use. Check out the release notes.
