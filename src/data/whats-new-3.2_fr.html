Les nouveautés de KLatexFormula 3.2

Cette fenêtre peut être ouverte en tous temps depuis l'affichage « A propos de KLatexFormula » accessible avec le bouton en mode étendu.

A présent une grande fenêtre de prévisualisation peut être ouverte après l'évaluation d'une formule (Raccourci: touche F2).

Un nouveau système de profils d'exportations Mime permet de préciser exactement quels formats vous aimeriez exporter lors d'opérations de Copier/Coller ou de Traîner/Déposer. Les formats Mime en question sont organisés en profils que vous pouvez sélectionner dans un menu sous les boutons TRAINER/COPIER.

A présent il est possible de rouvrir les images PNG préalablement sauvegardées (essayez de les traîner sur l'éditeur LaTeX).

L'éditeur détecte à présent les paquets LaTeX manquant lorsque vous écrivez un symbole pour lequel le paquet LaTeX correspondant n'a pas déjà été inclus dans le préambule.

A présent KLatexFormula convertit les polices en contours vectoriels dans les fichiers EPS et PDF, réglant certains problèmes d'exportation par ex. vers Adobe Illustrator.

La structure interne de la bibliothèque a été complètement réingénieurée pour permettre une plus grande flexibilité. Vous pouvez en lire plus sur le wiki.

Un petit greffon pour l'éditeur de KDE 4 permet une intégration agréable dans Kile, Kate et les autres applications qui utilisent KTextEditor.
