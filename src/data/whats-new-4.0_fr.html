Les nouveautés de KLatexFormula 4.0

Cette fenêtre peut être ouverte en tous temps depuis l'affichage « A propos de KLatexFormula » accessible avec le bouton en mode étendu.

KLatexFormula a été adapté pour Qt5 au lieu de Qt4, s'adaptant nativement aux écrans à haute résolution tels que les écrans Retina.

De nouveaux formats d'exportation sont disponibles: SVG via dvisvgm ou Inkscape, EMF/WMF fichier Metafile Windows via Inkscape, HTML encapsulant l'image, et AICB pour Adobe InDesign.

Retrouvez des exemples d'équations dans la bibliothèque › Ressources › Ouvrir la bibliothèque d'exemple. Les exemples incluent quelques équations courantes et un diagramme de Feynman.

Certains changement d'infrastructure du programme n'auront en principe aucun effet visible. Pour plus d'informations, consultez cette page.
